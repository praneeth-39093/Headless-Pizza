# 🍕 Headless-Pizza

A modern web-based pizza ordering interface built using HTML, CSS, and JavaScript. This project demonstrates a clean UI design and responsive layout for an online pizza ordering system.

---

## 🚀 Features

* 🍕 Attractive pizza menu display
* 🛒 Add to cart functionality (UI-based)
* 📱 Responsive design (works on mobile & desktop)
* 🎨 Clean and modern UI
* ⚡ Fast and lightweight

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
